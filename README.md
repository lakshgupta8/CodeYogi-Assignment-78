# XTodo

XTodo is a simple and efficient task management application designed to help you organize your daily activities. This project is part of the CodeYogi Assignment 78.

## Features

- **Add Todos**: Easily add new tasks to your list.
- **Toggle Completion**: Mark tasks as done or active.
- **Refresh List**: Reset your list to a default set of sample tasks.
- **Organized Views**: View your pending and completed tasks in separate sections.

## Tech Stack

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

## Setup Instructions

To run this project locally, follow these steps:

1.  **Install Dependencies**

    ```bash
    bun install
    ```

2.  **Start Development Server**
    ```bash
    bun run dev
    ```
